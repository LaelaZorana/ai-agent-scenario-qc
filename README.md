# ai-agent-scenario-qc

Tiny side project: a Python tool to QC the JSON scenarios used to train AI agents.

WIP. Goals:
- load a scenario JSON
- validate structure (JSON schema)
- detect logical defects (undeclared tools, undefined personas, etc.)
- score against a rubric
- emit a defect report

Inspired by the AI Agents course at Vanderbilt I just finished.
